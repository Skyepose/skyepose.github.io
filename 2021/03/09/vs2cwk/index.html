我们的群组有何不同？

一直以来，金融这个行业依托的是信息差来实现所谓的“盈利”，但是目前看来这条道路的在业内无法走得很远。在这里，我们希望拉近客户与机构之间的距离，让一种全新的互动讨论形式成为业界新的投资模式。在这里的每一份投资文档都是在不断根据时代的变化而不断更新进步的，越来越多的业界从业人员贡献自己的知识，就像开源的社区 Github 一样，通过这个纠正不正确的投资价值观，倡导理性投资回归。同时，我们也有多个 Telegram 讨论群组，并且会定期举办 Clubhouse 以及各类线上演讲来与大家沟通。
